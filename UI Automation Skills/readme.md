# Skill
## 基本信息 
- ID: taobao_ui_automation
- Name：淘宝UI自动化

## 功能描述 
自动完成电商网站登录、搜索、筛选、加购的任务

## 执行流程 
- 1.启动浏览器并访问目标电商网站 （https://automationexercise.com）
  （绕不过淘宝反爬人机验证，所以用公开电商网站）
- 2.等待用户人工登录并校验登录状态 
- 3.自动搜索指定商品
- 4.按条件筛选并加入购物车 
- 5.记录执行步骤并返回结构化结果

## 输入 
无额外参数输入，依赖用户在页面完成手动登录 

## 输出 
- status: 成功/失败
- message: 结果描述
- steps: 执行步骤列表
## 效果
- 可在飞书人机交互界面与大模型正常对话，大模型可判断是否闲聊或进行UI自动化任务。
- 执行中能准确打开无反爬电商网站（https://automationexercise.com），
能准确点击导航栏“Login”进入登录界面，能一直等待并循环检测是否登录成功（导航栏是否出现“Logout”），
能准确点击导航栏“product”进入商品页面，能准确点击搜索框输入“Tshirt”进行搜索，
能获取每件商品的价格并判断是否小于1000，能准确点击加入购物车按钮“Add to cart”，
能准确点击导航栏“Cart”进入购物车展示自动加购情况。
